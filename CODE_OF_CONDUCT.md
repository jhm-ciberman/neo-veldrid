# Contributor Covenant Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

Please read the full text at the link above. In summary: be respectful, be constructive, and be welcoming.

## AI-Assisted Contributions

Using AI tools (LLMs, code assistants, agents) is welcome. However, the contributor is fully responsible for the code they submit, regardless of how it was produced. All AI-generated code must be manually reviewed and tested by the contributor before submission.

Low-effort AI-generated content (PR spam, AI slop, bulk-generated issues) will be closed and repeat offenders may be banned.

## Enforcement

If you experience or witness unacceptable behavior, please report it by opening a [GitHub Issue](https://github.com/jhm-ciberman/neo-veldrid/issues).

Project maintainers are responsible for enforcing this code of conduct and may take appropriate action in response to any behavior they deem inappropriate.
